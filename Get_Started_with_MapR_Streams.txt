For MapR Streams
================

1. Create a stream:

[root@maprdemo mapr]# maprcli stream create -path /sample-stream_1

2. Create two topics:

[root@maprdemo mapr]# maprcli stream topic create -path /sample-stream_1  -topic fast-messages_1
[root@maprdemo mapr]# maprcli stream topic create -path /sample-stream_1  -topic summary-markers_1

3. List the topics:

[root@maprdemo mapr]# maprcli stream topic list -path /sample-stream_1

4. Get more info for the topic in JSON:

maprcli stream topic list -path /sample-stream_1 -json

5. Produce to the topic:

[root@maprdemo bin]# ./kafka-console-producer.sh --broker-list 127.0.0.1:9092 --topic /sample-stream_1:fast-messages_1

6. Consume from the topic:

[root@maprdemo bin]# ./kafka-console-consumer.sh --bootstrap-server 127.0.0.1:9092 --topic /sample-stream_1:fast-messages_1
